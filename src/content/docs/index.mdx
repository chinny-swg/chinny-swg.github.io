## What this is

[SWGEmu](https://www.swgemu.com/) recreates **Pre-CU** *Star Wars Galaxies* (the era before the 2005 Combat Upgrade) via a reverse-engineered server emulator. **Core3** is the server software — a C++ engine plus Lua game scripts.

These docs cover the **`chinny-swg`** fork of Core3 (with custom **SWGRealmsAPI** work): how to build it, deploy it with Docker, run it on a home server, and administer it.
